Cool Python Commands and Their Use Cases

enumerate() – Get index with items in a loop

for i, item in enumerate(['a', 'b', 'c']):
    print(i, item)
# Output:
# 0 a
# 1 b
# 2 c

zip() – Combine multiple iterables

names = ['Alice', 'Bob']
scores = [90, 85]
for name, score in zip(names, scores):
    print(f"{name} scored {score}")
# Output:
# Alice scored 90
# Bob scored 85

list comprehension – Concise loop for lists

squares = [x*x for x in range(5)]
print(squares)
# Output: [0, 1, 4, 9, 16]

lambda – Small anonymous functions

add = lambda x, y: x + y
print(add(2, 3))
# Output: 5

map() – Apply function to iterable

nums = [1, 2, 3]
doubled = list(map(lambda x: x * 2, nums))
print(doubled)
# Output: [2, 4, 6]
